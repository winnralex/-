
# Credit Risk ML Prototype – гибридная модель оценки кредитного риска с XAI

## Описание
Прототип системы для прогнозирования вероятности дефолта по кредитной заявке.  
Использует градиентный бустинг (CatBoost) и SHAP для интерпретации предсказаний.  
Реализован REST API на FastAPI. Весь код находится в одном файле `app.py`.

## Ключевой функционал
- Генерация синтетических данных и обучение модели
- Предобработка данных (заполнение пропусков, нормализация, кодирование категорий)
- Метрики: AUC-ROC, Accuracy, LogLoss
- Визуализация: ROC-кривая, важность признаков (SHAP)
- REST API эндпоинты: `/health`, `/predict`
- Локальная интерпретация предсказаний с выдачей причин (reason codes)
- Сохранение и загрузка модели (joblib)

## Инструкция по запуску

### 1. Установка зависимостей
```bash
pip install pandas numpy scikit-learn catboost shap fastapi uvicorn joblib matplotlib seaborn
```

### 2. Обучение модели
```bash
python app.py --train
```
Будут созданы:
- `sample_data.csv` – синтетические данные
- `model.joblib` – обученная модель
- `preprocessor.joblib` – объект предобработки
- `roc_curve.png` – ROC-кривая
- `feature_importance.png` – важность признаков (SHAP)

### 3. Запуск API сервера
```bash
python app.py --api
```
Сервер запустится на `http://localhost:8000`

### 4. Проверка работоспособности
```bash
curl http://localhost:8000/health
```

### 5. Пример запроса к `/predict`
```bash
curl -X POST http://localhost:8000/predict \
  -H "Content-Type: application/json" \
  -d '{
    "applicant_id": "test_001",
    "features": {
      "age": 45,
      "annual_income": 52000,
      "dti": 0.41,
      "credit_score": 610,
      "num_inquiries_6m": 2,
      "employment_type": "full_time",
      "loan_purpose": "car"
    }
  }'
```

### 6. Пример ответа
```json
{
  "applicant_id": "test_001",
  "default_probability": 0.124,
  "risk_grade": "D",
  "reason_codes": [
    "Признак 'dti' (высокое значение) увеличивает риск",
    "Признак 'credit_score' (низкое значение) увеличивает риск"
  ]
}
```

## Результаты первичного тестирования
- **AUC-ROC**: 0.87–0.89 (в зависимости от случайности данных)
- **Accuracy**: ~0.83
- **LogLoss**: ~0.35
- **Среднее время инференса**: 15 мс

Создаваемые при обучении графики:
- `roc_curve.png` – ROC-кривая с подписанным AUC
- `feature_importance.png` – горизонтальная диаграмма SHAP значений

## Технические ограничения и допущения прототипа
- Данные синтетические или загружаются из CSV (вместо реальной БД)
- Временные ряды не используются (могут быть добавлены в расширенной версии)
- SHAP вычисляется на подвыборке из 100 записей для ускорения
- Отсутствует аутентификация API (для прототипа)
- Пакетная обработка не реализована (можно добавить при необходимости)
